# kaikai_python_tools
## shutil_images.py
   ``python shutil_images.py -i 参数1 -s 参数2 -o 参数3``
   > 例如在本程序中，参数1表示打开的文件名，参数2表示要拷贝的文件夹路径，参数3表示处理后输出到的文件夹。
   利用命令行参数批量处理文件，具体内容是根据txt文本文件里的名字找到对应的图片，将其放在新的文件夹下。
